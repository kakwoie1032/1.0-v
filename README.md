# 🏫 학급 종합 관리 프로그램 (Class Hub) MVP

선생님과 학생을 실시간으로 연결해 주는 고성능 **Vanilla JS + Tailwind CSS + Firebase** 기반 학급 종합 통합 관리 플랫폼입니다.

이 프로그램은 별도의 프레임워크나 복잡한 백엔드 구축 없이도, 순수 웹 표준 기술과 구글 파이어베이스(Firebase) 서비스를 결합하여 안정적이고 쾌적한 디지털 학급 환경을 구축할 수 있도록 제작되었습니다.

---

## 🌟 주요 핵심 기능

### 1. 👑 학교 관리자 (Admin Master)
* **학급 편성 및 조직 구성**: 임의의 학년과 학급(예: 1학년 1반, 2학년 3반 등)을 생성 및 철회할 수 있습니다.
* **선생님 계정 발급 및 부여**: 담임 교사 자격을 부여할 수 있는 교직원용 디렉터리를 조율합니다.
* **학교 전체 공지 발행**: 학교 전체 학생에게 일괄 노출되는 중요한 학교 공지사항을 발행합니다.

### 2. 🧑‍🏫 담임 선생님 (Teacher Control Panel)
* **우리 반 학생 명부 전장**: 학생 추가, 정렬, 성명 편집 및 고유 학번 조율이 가능합니다.
* **1인 1역 & 서기 지정**: 모든 학생에게 고유한 학급 역할을 할당하며, 반을 대리 관리할 수 있는 **'학급 서기'** 특권을 부여합니다.
* **우리 반 급식표 & 시간표 제어**: 요일별 교시별 주간 시간표와 날짜별 급식 식단을 갱신, 관리(이미지 업로드 대응 가능)할 수 있습니다.
* **게시판 중재 및 조정**: 학급 소통글 중 부적절한 게시글에 대해 강제 삭제 혹은 블라인드(숨김) 처리를 적용할 수 있습니다.

### 3. 👦 학급 학생 (Student Dashboard)
* **종합 상황 대시보드**: 오늘 배포된 학교 공지, 우리 반 교실 알림, D-Day 카운트다운 일정, 그리고 오늘의 영양 급식 식단을 원페이지로 모니터링합니다.
* **열품타 스타일 학습 타이머 (Stopwatch)**: 실시간 공부 시간을 누적 측정하며, 기록 저장 시 과목 태그와 학습 다짐 메모를 첨부할 수 있습니다.
* **실시간 공부 시간 랭킹**: 오늘/주간/월간 단위로 학급 구성원들의 공부시간 누적 순위표(🥇 🥈 🥉 메달 시상)가 집계되어 선의의 학업 경쟁을 유도합니다.
* **학급 소통 게시판**: 자유/질문/수행평가/준비물/건의사항 카테고리별로 자유롭게 의사소통하고 공감(좋아요) 및 댓글 릴레이를 펼칠 수 있습니다.

### 4. 📜 학급 서기 (Secretary Workspace)
* **서기 전용 집무실 활성화**: 학생용 종합 대시보드를 기본으로 이용하되, 특별 임무용 전용 탭이 해제됩니다.
* **학급 디데이 개설/삭제**: 반 전체 주요 시험 일정, 모의고사, 수행평가 마감일 등 카운트다운 일정을 관리합니다.
* **학급 공지사항 작성**: 우리 반에 국한된 긴급 가정통신문이나 알림장을 손쉽게 유포할 수 있습니다.

## 💾 듀얼 모드 데이터 저장소 (Dual-Mode Persistence)

본 시스템은 **어떠한 서버 지연 없이 즉각 작동하는 오프라인 MVP 테스트 환경**과 **실제 상용 Firebase 클라우드** 환경을 동시에 제공합니다.

1. **임시 데모 모드 (Local Storage Mocking)**:
   * 최초 부팅 시, 혹은 별도의 자격증명을 입력하지 않았을 경우 작동합니다.
   * 브라우저의 `localStorage` 공간을 모형 Firestore처럼 흉내 내어 모든 회원가입, 로그인, 게시판 작성, 시간표 조율 등이 안정적으로 완벽 작동합니다.
2. **실제 클라우드 파이어베이스 연동 (Cloud Run Firebase)**:
   * 우측 상단의 `⚙️ Firebase 설정` 단추를 누른 뒤, 구글 파이어베이스 콘솔에서 발급받은 `firebaseConfig` JSON 객체를 붙여넣고 저장하면 즉시 실시간 데이터 교환이 가능합니다.

## 📦 실제 구글 파이어베이스(Firebase) 설치 및 서비스 배포 가이드

본 프로그램을 실제 개인 또는 학교 Firebase 계정에 연결하여 전교생 및 반 학생들과 영구 서비스하려면 아래 가이드를 준수해 주십시오.

### STEP 1: Firebase 프로젝트 및 웹 앱 생성
1. [구글 파이어베이스 콘솔](https://console.firebase.google.com/)에 접속하여 신규 프로젝트를 만듭니다.
2. 프로젝트 대시보드에서 `웹(Web)` 플랫폼 앱을 추가 등록합니다.
3. 앱 추가 완료 시 노출되는 `firebaseConfig` 코드 블록을 복사해 둡니다.

### STEP 2: 서비스 기능 활성화
1. **Authentication**: `이메일/비밀번호` 로그인 공급업체를 활성화합니다.
2. **Cloud Firestore**: 테스트 모드 또는 프로덕션 모드로 데이터베이스 생성을 완료합니다.
3. **Cloud Storage**: 프로필 및 급식, 공지용 이미지 저장 버킷을 활성화합니다.

### STEP 3: 보안 규칙 적용 (Security Rules Protection)
프로젝트 루트 폴더에 발급된 파일들을 파이어베이스 콘솔 내 각 규칙 탭에 업로드하여 소중한 학급 개인정보 및 학력 기록을 철저히 암호화 차단해 주십시오.

* **Firestore 보안 규칙**: `firestore.rules` 파일 내용을 복제 적용합니다.
* **Storage 보안 규칙**: `storage.rules` 파일 내용을 복제 적용합니다.
